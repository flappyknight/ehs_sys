# 企业权限管理模块 (Enterprise Permission Management)

## 模块概述

企业权限管理模块负责角色定义、权限分配和审批流程配置。

## 主要功能

### 1. 角色管理
- 创建自定义角色
- 编辑角色信息
- 删除角色
- 角色权限配置

### 2. 权限管理
- 查看权限列表
- 权限分配
- 权限回收

### 3. 审批流程配置
- 配置审批流程
- 设置审批人
- 审批级别设置

## 权限要求

- **企业管理员**: 可以管理所有角色和权限
- **其他用户**: 只能查看
