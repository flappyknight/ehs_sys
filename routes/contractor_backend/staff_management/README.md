# 承包商人员管理模块 (Contractor Staff Management)

## 模块概述

承包商人员管理模块负责承包商内部员工的管理功能。

## 主要功能

### 1. 员工管理
- 添加承包商员工
- 查看员工列表
- 编辑员工信息
- 禁用/启用员工账户

### 2. 资质管理
- 员工资质证书管理
- 资质到期提醒
- 资质审核

### 3. 培训记录
- 培训记录管理
- 培训证书上传
- 培训有效期管理

## 权限要求

- **承包商管理员**: 可以管理所有员工
- **作业人员**: 只能查看自己的信息
