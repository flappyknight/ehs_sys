# 快速开始指南

## 🚀 立即开始使用新功能

### 1️⃣ 启动服务

#### 启动后端
```bash
cd /Users/dubin/work/ehs_sys
./start-server.sh
```

#### 启动前端
```bash
cd /Users/dubin/work/ehs_sys/web
npm run dev
```

### 2️⃣ 访问系统

打开浏览器访问：**http://localhost:5173/login**

### 3️⃣ 体验新功能

#### 查看登录页面
- 可以看到新增的"身份选择"下拉框
- 可以看到"立即注册"按钮

#### 注册新用户
1. 点击"立即注册"按钮
2. 选择身份类型（企业或承包商）
3. 填写表单信息
4. 点击"注册"按钮
5. 等待3秒自动跳转到登录页面

#### 登录系统
1. 选择对应的身份类型
2. 输入用户名和密码
3. 点击"登录"按钮
4. 成功后跳转到仪表板

## ❓ 常见问题

### Q1: 注册时提示"用户名已存在"
**A**: 请更换一个新的用户名，用户名必须是唯一的。

### Q2: 注册时提示"两次输入的密码不一致"
**A**: 请确保"密码"和"确认密码"输入的内容完全一致。

### Q3: 注册时提示"请输入有效的手机号"
**A**: 请输入有效的中国手机号格式，如：13800138000

### Q4: 系统管理员如何注册？
**A**: 系统管理员账号不支持自助注册，需要联系系统管理员创建。

### Q5: 注册成功后如何登录？
**A**: 注册成功后会自动跳转到登录页面，使用刚注册的用户名和密码登录即可。

## 🔧 故障排查

### 前端无法访问
1. 检查前端服务是否启动：`npm run dev`
2. 检查端口5173是否被占用
3. 查看浏览器控制台是否有错误

### 后端无法访问
1. 检查后端服务是否启动：`./start-server.sh`
2. 检查端口8100是否被占用
3. 查看日志文件：`tail -f server.log`

### 数据库连接失败
1. 检查数据库是否启动：`docker ps`
2. 启动数据库：`./start-docker-db.sh`
3. 检查数据库配置：`config.py`
