# 工单浏览模块 (Ticket View)

## 模块概述

工单浏览模块允许承包商查看和处理相关的作业工单。

## 主要功能

### 1. 工单查看
- 查看相关工单列表
- 查看工单详情
- 按状态筛选工单

### 2. 工单执行
- 工单执行情况上报
- 上传作业照片
- 工单完成确认

### 3. 工单统计
- 工单统计分析
- 执行情况报表

## 权限要求

- **承包商管理员**: 可以查看所有相关工单
- **作业人员**: 可以查看分配给自己的工单
