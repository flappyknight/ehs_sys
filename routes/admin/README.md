# 系统账户后台模块 (Admin Backend Module)

## 模块概述

系统账户后台模块负责系统管理员的核心管理功能，包括用户认证、系统级用户管理、企业管理、承包商管理等系统级别的操作。

## 主要功能

### 1. 认证管理
- 用户登录/登出
- Token 生成与验证
- 密码管理

### 2. 系统用户管理
- 管理员账户管理
- 查看所有用户
- 用户权限管理

### 3. 企业管理
- 创建/查看/编辑/删除企业
- 企业信息维护
- 企业状态管理

### 4. 承包商管理
- 创建/查看/编辑/删除承包商
- 承包商信息维护
- 承包商资质管理

## 权限要求

- 所有接口均需要管理员权限 (UserType.admin)
- 部分接口对企业管理员开放

## 技术栈

- FastAPI
- SQLModel
- JWT 认证
- 密码加密 (bcrypt)
