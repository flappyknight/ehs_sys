# 企业作业流程管理模块 (Enterprise Workflow Management)

## 模块概述

企业作业流程管理模块负责作业计划的管理、审批和人员签到等功能。

## 主要功能

### 1. 计划管理
- 查看作业计划列表
- 查看计划详情
- 审批作业计划

### 2. 参与人员管理
- 查看计划参与人员
- 人员签到管理
- 人员资质验证

### 3. 计划执行跟踪
- 计划执行状态
- 签到情况统计
- 执行进度跟踪

## 权限要求

- **企业管理员**: 可以审批和管理所有计划
- **现场人员**: 可以查看计划和管理签到
- **普通员工**: 只能查看相关计划
