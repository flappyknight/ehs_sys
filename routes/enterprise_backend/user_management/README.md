# 企业用户管理模块 (Enterprise User Management)

## 模块概述

企业用户管理模块负责企业内部员工、部门和厂区的管理功能。

## 主要功能

### 1. 员工管理
- 添加企业员工
- 查看员工列表
- 编辑员工信息
- 禁用/启用员工账户
- 员工角色分配

### 2. 部门管理
- 创建部门
- 查看部门列表
- 编辑部门信息
- 删除部门
- 查看部门成员

### 3. 厂区管理
- 创建厂区
- 查看厂区列表
- 编辑厂区信息
- 删除厂区
- 厂区与部门关联

## 权限要求

- **企业管理员**: 可以管理所有员工、部门和厂区
- **现场人员**: 可以查看员工和厂区信息
- **普通员工**: 只能查看自己的信息

## 数据隔离

- 企业用户只能管理自己企业的数据
- 通过 `enterprise_id` 自动过滤数据
- 跨企业访问被拒绝
