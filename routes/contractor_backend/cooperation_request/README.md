# 合作申请管理模块 (Cooperation Request Management)

## 模块概述

合作申请管理模块负责处理企业发起的合作邀请。

## 主要功能

### 1. 合作邀请查看
- 查看合作邀请列表
- 查看邀请详情

### 2. 合作申请处理
- 接受合作邀请
- 拒绝合作邀请
- 填写合作信息

### 3. 合作项目查看
- 查看当前合作项目
- 查看合作历史

## 权限要求

- **承包商管理员**: 可以处理合作申请
- **作业人员**: 只能查看
