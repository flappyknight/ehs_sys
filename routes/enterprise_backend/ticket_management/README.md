# 企业工单管理模块 (Enterprise Ticket Management)

## 模块概述

企业工单管理模块负责作业工单的创建、审批、跟踪和管理。

## 主要功能

### 1. 工单管理
- 创建作业工单
- 查看工单列表
- 编辑工单信息
- 删除工单

### 2. 工单审批
- 工单审批流程
- 多级审批
- 审批记录查询

### 3. 工单跟踪
- 工单状态跟踪
- 工单执行情况
- 工单完成确认

## 权限要求

- **企业管理员**: 可以管理所有工单
- **现场人员**: 可以创建和查看工单
- **普通员工**: 只能查看相关工单
