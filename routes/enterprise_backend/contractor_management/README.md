# 企业承包商管理模块 (Enterprise Contractor Management)

## 模块概述

企业承包商管理模块负责企业与承包商的合作关系管理，包括承包商信息维护、合作项目管理等。

## 主要功能

### 1. 承包商管理
- 查看合作承包商列表
- 查看承包商详情
- 承包商评价与评分

### 2. 合作项目管理
- 创建与承包商的合作项目
- 查看项目列表
- 编辑项目信息
- 结束项目

### 3. 承包商创建
- 创建新承包商并建立合作
- 邀请已有承包商合作

## 权限要求

- **企业管理员**: 可以管理所有承包商和项目
- **现场人员**: 可以查看承包商和项目信息
- **普通员工**: 无权限

## 数据隔离

- 企业只能看到与自己有合作关系的承包商
- 通过项目关联实现数据隔离
